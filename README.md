This ESP32 project is a simple conveyor-style item counting and monitoring system that uses an IR sensor, LEDs, a buzzer, and a reset button. The IR sensor detects when an object passes through and increases the item count by one each time a new object is detected. A green LED indicates normal operation when an item is being detected, while a red LED and buzzer are activated if no object is detected for more than 10 seconds, signaling an idle or alert condition. A push button is included to reset the item count back to zero. Overall, the system provides real-time object detection, counting, and status indication for basic automation or industrial monitoring applications.
